# wt-tracker
High-performance WebTorrent tracker
## Build instructions

```sh
npm install
npm run build
```

## Run instructions

```sh
node dist/run-uws-tracker.js [config.json]
```

or

```sh
node start [config.json]
```

## Configuration

See [config.json](sample/config.json).
